### 🛠️ Portfolio & Projects

#### 📡 Space Systems & Network Simulations
*   **[SPICE-ns-Project](https://github.com/PandiaJason/SPICE-ns-Project)**: 📡 SPICE-ns — An Extension for the ns-3 network simulator to simulate DSN (Deep Space Network) and DTN (Delay-Tolerant Networking) communications.
*   **[qubo-space-routing](https://github.com/PandiaJason/qubo-space-routing)**: Mathematical routing optimization in space environments using Quadratic Unconstrained Binary Optimization (QUBO).
*   **[ns3](https://github.com/PandiaJason/ns3)**: Core simulator configurations and modifications for planetary network architectures.
*   **[DSN](https://github.com/PandiaJason/DSN)**: Open-source learning log and technical resources for Deep Space Networking.
*   **[classic](https://github.com/PandiaJason/classic)**: *Ranotot Classic* — A 2D space cargo delivery web game featuring planet gravity mechanics.

#### 🦀 AI Infrastructure & System Programming
*   **[nanos](https://github.com/PandiaJason/nanos)**: Rust-based, hardware-isolated WebAssembly micro-runtime for AI agents with Apple Metal/CUDA GPU offload. Optimized for extreme performance (<15MB RAM, <50ms boot).
*   **[MidDB](https://github.com/PandiaJason/MidDB)**: A lightweight, AI-aware hybrid database prototype in C++. Combines traditional structured data storage with vector embeddings for semantic query memory.
*   **[amphitude](https://github.com/PandiaJason/amphitude)**: C++ software library for high-speed digital audio and signal processing.
*   **[AI-Chatbot-using-Deep-Learning-based-NLP](https://github.com/PandiaJason/AI-Chatbot-using-Deep-Learning-based-NLP)**: A Transformer-based generic chatbot framework.

#### 🔗 Web3 & Smart Contracts
*   **[Proof-of-Knowledge-On-Chain](https://github.com/PandiaJason/Proof-of-Knowledge-On-Chain)**: Solidity-based implementation of Non-Transferable Non-Fungible Tokens (Soulbound Tokens) for the PoKoC credential system.
*   **[dvault-docs](https://github.com/PandiaJason/dvault-docs)**: Smart contract wallet architectures using ERC-4337 Account Abstraction.
*   **[contrackts-docs](https://github.com/PandiaJason/contrackts-docs)**: End-to-end traceability using blockchain multi-party registries.
*   **[jsncrypts-docs](https://github.com/PandiaJason/jsncrypts-docs)**: Decentralized publishing and subscription models.
